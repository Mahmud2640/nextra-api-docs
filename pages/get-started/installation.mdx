## Common Headers

**This is Common Header**

| **Header**          | **Value**             | **Required** |
| :------------------- | :-------------------: | -----------: |
| Content-Type    | `application/json`    | Yes     |
| Accept-Encoding  | `gzip, deflate`       | Yes      |
| Authorization   | `Bearer <your-token>` | Yes     |
